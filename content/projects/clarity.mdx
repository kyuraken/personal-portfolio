### Clarity

Clarity is a full-stack personal finance dashboard that integrates the Plaid API
to aggregate and display real-time transaction data across multiple linked bank
accounts.

**Features**

- **Real-Time Transactions**: Aggregates transaction data across multiple linked
  bank accounts using the Plaid API.
- **Anomaly Detection**: Statistical anomaly detection system flags unusual
  spending patterns, alerting users to potential fraudulent or out-of-budget
  transactions.
- **RESTful Backend**: Node.js and Express backend with MySQL for persisting user
  accounts, transaction history, and alert configurations.
- **Secure Authentication**: JWT-based authentication with user-scoped access
  control across all API endpoints.
